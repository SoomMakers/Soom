  
# Soom


###  How to run the code

You can check the app on the local server. 
Please follow the steps below.

### 1. Install

```bash
npm install
```

### 2. Run

```bash
npm start
```
open [http://localhost:3000](http://localhost:3000/) to view it in thre browser.

----

#### How to run tests

```bash
npm test
```
